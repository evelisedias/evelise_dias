Sobre mim

Me chamo Evelise, sou Desenvolvedora Web. Sou apaixonada por música e basicamente tudo que envolve tecnologia.


Atualmente trabalho como ISM na Saipos | Sistema para Restaurante em tempo integral. Nessa função, além de gerenciar a implantação de novos clientes, tenho a oportunidade de entender profundamente as necessidades dos usuários, o que enriquece minha visão como desenvolvedora.


Sou graduanda em Análise e Desenvolvimento de Sistemas pela Unisinos, e tenho facilidade em trabalhar com JAVA, pois foi a primeira tecnologia de backend que aprendi. Desde julho de 2024, estou expandindo meus horizontes através de cursos na Alura, focando em JavaScript para fortalecer minha base de aprendizado. Meu objetivo é, após dominar essa linguagem, mergulhar em tecnologias como Node.js e React, que estão na minha lista de próximos desafios.

Projetos

Nos últimos meses desenvolvi alguns projetos trabalhando com HTML, CSS e Javascript outros usando Java. Você pode encontrar todos os meus repositórios no GitHub.

Laço Amarelo

O Laço Amarelo foi desenvolvido como parte da avaliação final de uma disciplina na faculdade. O projeto foca na saúde mental, especificamente na bipolaridade. Apesar do acesso à informação facilitado nos dias de hoje, ainda há muita desinformação sobre essa condição, que afeta aproximadamente 140 milhões de pessoas globalmente, de acordo com a OMS. O objetivo foi criar uma plataforma informativa e acessível para ajudar as pessoas que possuem esse diagnostico.

Development Tools
  • HTML/CSS: Para a estruturação e estilização da página.
  • JavaScript: Implementação de interatividade, como carrosséis de notícias.

Podflix

Podflix é uma plataforma de podcast que recria experência de streaming, similar à Netflix, mas focada exclusivamente para podcast. Desenvolvido como um projeto de orientação da DIO (Digital Innovation One), o podflix permite que o usuário naveguem e ouça alguns podcasts de forma intuitiva e envolvente.

Development Tools
  • HTML/CSS: Para a estruturação e estilização e responsividade da página
  • JavaScript: Implementação de interatividade
  • OWL Carousel JS: Para dar movimento e interatividade aos banner dos podcasts

Conversor de Moedas

Este projeto foi desenvolvido para aprender a realizar requisições a APIs. O conversor possui uma interface simples e intuitiva, permitindo ao usuário converter valores entre Dolar, Real e Euro de maneira rápida e fácil.

Development Tools
  • HTML/CSS: Para a estruturação e estilização da página.
  • JavaScript: Lógica para geração de senhas aleatórias e personalização.
  • JQuery

Gerador de Senhas

Desenvolvi este projeto pessoal para aprofundar meu conhecimento em métodos JavaScript. O gerador de senhas apresenta uma interface amigável e permite criar senhas seguras com facilidade usando os requisitos conforme a necessidade do cliente.

Development Tools
  • HTML/CSS: Para a estruturação e estilização da página.
  • JavaScript: Lógica de conversão de moedas e manipulação de DOM.
